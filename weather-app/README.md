# Weather App - Glass Morphism

A modern weather web application built with React, Vite, Tailwind CSS, and Framer Motion. Features current weather, historical data, and marine conditions with a beautiful glass morphism design.

## Setup

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Navigate to project
cd weather-app

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```

## Features

- **Current Weather** - Real-time weather for any location
- **Historical Weather** - Past weather data with date picker
- **Marine Weather** - Ocean conditions by latitude/longitude
- **Location Search** - Search with dropdown suggestions
- **Glass Morphism UI** - Modern dark theme with glass effects
- **Responsive** - Mobile-first design

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Axios
- Framer Motion

## API

Uses [WeatherStack API](https://weatherstack.com/). API key is configured in `src/services/api.js`.
